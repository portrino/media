Generate a bunch of thumbnails in advance to speed up the output of the Media BE module.

This process can take a lot of time. For the first time, prefer to run the CLI command:

cd {sitePath};
./typo3/cli_dispatch.phpsh extbase thumbnail:generate

# Available options:
--limit --offset --configuration --verbose
Tip! You can also configure a Scheduler Task for that. {namespace v=Fab\Vidi\ViewHelpers}
